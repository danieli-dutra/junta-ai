# Documentação Técnica — Grana.ai

Esta pasta reune toda a documentação técnica utilizada durante o desenvolvimento do Grana.ai.

A documentação está organizada em quatro grandes áreas:

- **AI** — Especificação do agente financeiro, comportamento, memória, ferramentas e regras de negócio.
- **Architecture** — Arquitetura do sistema, integração entre frontend, backend, banco de dados e MCP.
- **API** — Documentação dos endpoints REST e contratos de comunicação.
- **Product** — Visão do produto, roadmap, decisões técnicas e regras de negócio.

O objetivo desta documentação é servir como referência para desenvolvimento, manutenção e evolução do projeto.
